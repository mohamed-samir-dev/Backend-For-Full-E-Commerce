# E-Commerce Backend

Backend API for e-commerce platform.

## Tech Stack

- Node.js
- Express.js
- MongoDB

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
JWT_EXPIRE=7d
```

## Run

```bash
npm start
```
